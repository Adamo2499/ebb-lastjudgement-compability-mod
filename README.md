## Enhanced Boss Bars icons for Fall From Grace
### Description

This mod adds [Enhanced Boss Bars](https://steamcommunity.com/workshop/filedetails/?id=2635267643) icons for [Last Judgement](https://steamcommunity.com/sharedfiles/filedetails/?id=3343815332])

### Installation
#### Steam Workshop
Get it on [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3658326496)

#### NonSteam launchers + cracked
1. Download code by going to Code -> Download ZIP
2. Extract ebb-lastjudgement-compability-mod-main.zip into Isaac's mods folder and remove -main from extracted folder's name
3. Launch the game and enjoy!
